## Installation

### 1. Clone project by using ```git clone```
```
$ git clone git@github.com:thaotpphung/stockapp.git
$ cd stockapp
```

### 2. Install MongoDB
[MongoDB Download Page](https://docs.mongodb.com/manual/administration/install-community/)

### 3. Run application

```
$ npm install
$ node app.js
```
